# 🌍 Great for the Planet

A React-based web application focused on raising awareness about carbon emissions and helping users understand their environmental impact.

---

## 🔗 Live Demo
https://ryhart07.github.io/greatfortheplanet

---

## 🚀 Features

* 📊 **Live Carbon Intensity Data** (via API)
* 🌱 **Region-based insights** into emissions
* 📖 **User-friendly explanations** of complex data
* 🎯 Clean, responsive UI
* 🔄 Refactored from a static HTML/CSS/JS project into React

---

## 🛠️ Built With

* React (Vite)
* JavaScript (ES6+)
* CSS
* React Router
* External API (Carbon Intensity Data)

---

## 📂 Project Structure

```
src/
  components/
  pages/
  hooks/
  services/
  styles/
```

---

## 🧠 What I Learned

* Converting a static website into a React application
* Using `useState` and `useEffect` for dynamic data
* Fetching and displaying API data
* Improving user experience with interactive UI elements
* Structuring a React project using components, hooks, and services

---

## 🔧 Installation

1. Clone the repository:

```
git clone https://github.com/ryhart07/greatfortheplanet.git
```

2. Install dependencies:

```
npm install
```

3. Run the development server:

```
npm run dev
```
